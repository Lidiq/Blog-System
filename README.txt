Blog System:
-creates a blog for the user on register
-the user can:
   write/edit/delete articles;
   categorize articles;
   allow/disallow comments;
   choose a theme;
   adding comments;
   change his profile;
-if not registered, people can:
   read the articles;
   adding comments if it is allowed;
-administrator can delete users
